# Priorität

| Priorität | Bereich                           | Warum                                                   |
| --------: | --------------------------------- | ------------------------------------------------------- |
|         1 | Mausrad-Funktionen                | Auf Touch-Geräten gar nicht vorhanden                   |
|         2 | Rechtsklick-Auswahl               | Auf Touch-Geräten nicht natürlich verfügbar             |
|         3 | Canvas-Drag/Panning               | Funktioniert mit Maus, aber nicht zuverlässig mit Touch |
|         4 | Zoom-Out per einfacher Linksklick | Muss wegen Tap-Konflikten neu gedacht werden            |
|         5 | UI-Controls                       | Funktionieren auf Tablet meist schon, daher zuletzt     |

# Aktuelle Events und Ersetzungen

#### Desktop:
- Linke Maustaste ziehen: Ansicht verschieben
- Rechte Maustaste halten: Zoom-Auswahl anzeigen
- Mausrad während Zoom-Auswahl: Auswahlrahmen skalieren
- Rechte Maustaste loslassen: hineinzoomen
- Doppelklick: herauszoomen
- Iterationstiefe: über Control-Panel

#### Touch:
- Ein-Finger-Ziehen: Ansicht verschieben
- Zwei-Finger-Spreizen: hineinzoomen
- Doppeltippen: herauszoomen
- Iterationstiefe: über Control-Panel
